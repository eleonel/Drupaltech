Setup and Installation
1.) Download and add to sites/all/modules
2.) Enable module and grant permission for "access spaces og clone" on user permissions page
3.) Navigate to spaces_og_clone or spaces_og_clone/{nid} if you want to link to it in views
4.) Select features from which content and from which roles you wish to clone, 
    set user who will be author of new nodes, and clone!
